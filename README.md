# Broadcast Clock

Static broadcast-safe clock for GitHub Pages with three modes: clock, countdown, and stopwatch.

## Features

- Fullscreen black background with 16:9 safe-area framing
- Large, high-contrast, tight-spaced `HH:MM:SS` display for confidence monitors
- Default timezone: `America/Vancouver`
- Modes: `clock`, `countdown`, `stopwatch`
- Countdown controls with **time input + Set button**
- URL params: `?tz=America/Vancouver`, `?mode=countdown&target=19:30`
- Keyboard shortcuts: `C`, `D`, `S`, `Space`, `R`
- No build tools or external dependencies
