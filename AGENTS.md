# Instructions for Codex

This is a simple GitHub Pages static site.

Workflow:
- Always start from the latest main branch.
- Create one branch per task.
- Create one pull request per task.
- Do not reuse old conflicted branches.
- Do not stack changes onto an existing PR unless explicitly asked.
- If a PR has conflicts, abandon it and create a fresh branch from latest main.
- Keep the app dependency-free: plain HTML, CSS, and JavaScript only.
- Update README.md when behavior changes.
- Stop after opening a clean PR and wait for review.
